# Task Manager App

A simple **Task Manager** built with **React**, **TypeScript**, and **localStorage**. This app allows users to **add, view, and manage tasks** with a login system. All tasks are stored locally in the browser.

Check out the **live demo** here: [Task Manager Live](https://task-manager-co.netlify.app)

## Features

- **User Login:** Simple login system to access your tasks.
- **Add Tasks:** Quickly create tasks with a title.
- **Mark Completed:** Toggle tasks as completed or pending.
- **Delete Tasks:** Remove tasks when done.
- **Persistent Storage:** Tasks are saved in the browser using `localStorage`.
- **TypeScript Support:** Strongly typed code for better development experience.
- **Responsive Design:** Works on desktop and mobile devices.

---

## Tech Stack

- **React** (with Hooks)
- **TypeScript**
- **Vite** (for fast project setup)
- **localStorage** (for data persistence)
